# AQITrackerApp
We can check air quality of any city around the world with this app
A modern, responsive web application that tracks the Air Quality Index (AQI) and 
Weather conditions for any city worldwide in real-time. 
This dashboard features a beautiful glassmorphism
UI, interactive charts, and dynamic color-coded indicators.
